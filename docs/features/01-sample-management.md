# 기능 명세 - 시료 관리

## 개요
시료(Sample)는 이 시스템의 가장 기본이 되는 단위다. 각 시료는 고유한 이름과 속성을 가지며,
시스템에 등록된 시료만 주문할 수 있다.

## 하위 기능

### 1. 시료 등록
새로운 시료를 시스템에 추가한다.

**속성 값**

| 항목 | 설명                                   |
|---|--------------------------------------|
| 시료 ID | 시료를 식별하는 고유 값 (예: `S-001`)           |
| 이름 | 시료명 (예: 실리콘 웨이퍼-8인치)                 |
| 평균 생산시간 | 시료 1개(ea)를 생산하는 데 걸리는 평균 시간 (sec/ea) |
| 수율 | 정상적으로 생산되는 비율                        |

**수율 계산식**
```
수율 = 정상적인 시료 수 / 총 생산 시료 수
예) 100개 생산 중 정상 물품 90개 → 수율 = 0.9
```

시료 등록 시 재고 수량은 입력받지 않으며, 초기 재고는 0이다. 이후 생산이 완료될 때마다
실생산량이 재고로 편입된다. 자세한 내용은 [05-production-line.md](05-production-line.md) 참고.

### 2. 시료 조회
등록된 모든 시료 목록을 확인한다. 목록에는 현재 재고 수량도 함께 표시한다.
5개씩 pagination을 하고 page 정보를 하단에 보여준다.

### 3. 시료 검색
이름 등 속성으로 특정 시료를 검색한다.

## 비즈니스 규칙
- 시스템에 등록되지 않은 시료는 주문할 수 없다.
- 시료 ID는 고유해야 한다.
