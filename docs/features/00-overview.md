# 기능 명세 - 개요 (메인 메뉴)

## 목적
프로그램 시작 시 기능별 메뉴를 표시하고, 전체 시료/주문 현황을 요약해서 보여준다.

## 메인 화면 구성
- 시스템 현재 시각
- 등록 시료 종수, 총 재고 수량
- 전체 주문 건수, 생산 라인 대기 건수
- 하위 기능 메뉴 목록

## 메뉴 목록

| 번호 | 메뉴 | 설명 | 관련 문서 |
|---|---|---|---|
| 1 | 시료 관리 | 신규 시료 등록, 목록 조회, 이름 검색 | [01-sample-management.md](01-sample-management.md) |
| 2 | 시료 주문 | 고객 주문 접수(예약) | [02-order-reservation.md](02-order-reservation.md) |
| 3 | 주문 승인/거절 | 접수된 주문의 승인/거절 처리 | [03-order-approval.md](03-order-approval.md) |
| 4 | 모니터링 | 상태별 주문 수, 시료별 재고 현황 확인 | [04-monitoring.md](04-monitoring.md) |
| 5 | 생산라인 조회 | 생산 중/대기 중인 주문 확인 (FIFO) | [05-production-line.md](05-production-line.md) |
| 6 | 출고 처리 | CONFIRMED 상태 주문 출고 | [06-shipment.md](06-shipment.md) |
| 0 | 종료 | 프로그램 종료 | - |

## 사용자 역할
- **주문 담당자**: 시료 주문(예약) 담당
- **생산 담당자**: 주문 승인/거절, 생산 관리, 출고 담당

## 참고
- 화면 구성(레이아웃, 색상, 표기 형식 등)은 자유롭게 결정한다. 원본 자료의 예시 UI는 이해를 돕기 위한 참고용이다.
