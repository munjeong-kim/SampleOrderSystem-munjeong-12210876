# S-Semi 반도체 시료 생산주문관리 시스템

## 개요
가상의 반도체회사 "S-Semi"의 반도체 시료(sample) 생산주문관리 시스템입니다.
고객으로부터 시료 생산을 주문받아 생산 및 납품까지의 과정을 관리하는 콘솔 기반 애플리케이션입니다.

## 기술 스택
- 언어: Python
- 테스트: pytest, pytest-mock

## 주요 도메인
- 고객(Customer) 주문 접수
- 시료(Sample) 생산 관리
- 납품(Delivery) 관리

## 개발 가이드
- 콘솔(CLI) 기반으로 동작하는 프로그램으로 구현합니다.
- 테스트 코드는 pytest 컨벤션을 따르며, 외부 의존성 mocking에는 pytest-mock을 사용합니다.
- 커밋 메시지는 한글로 작성합니다.
- 테스트 함수 이름은 한글로 작성합니다.
